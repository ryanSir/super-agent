## MODIFIED Requirements

### Requirement: 聊天界面整体布局
聊天界面 SHALL 采用三段式布局：顶部固定 Header、中间可滚动消息区、底部固定输入区。消息区域最大宽度 760px，水平居中，左右内边距 24px。

#### Scenario: 页面初始加载
- **WHEN** 用户打开应用
- **THEN** 显示顶部 Header（含标题、连接状态、新会话按钮）、空消息区（含快捷问题卡片）、底部输入区

#### Scenario: 消息区域滚动
- **WHEN** 消息数量超过可视区域高度
- **THEN** 消息区域出现滚动条，新消息自动滚动到底部，Header 和输入区保持固定

#### Scenario: 快捷问题卡片显示
- **WHEN** 消息列表为空且无正在处理的回复
- **THEN** 显示 2 列快捷问题卡片网格，点击后直接发送对应问题
